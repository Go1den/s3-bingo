How to Play

Watch this brief video for an explanation of how to play. Ultimately, you can play with whatever rules you prefer. These are suggestions.

Some suggested ways to play:

Team Comp Bingo (Easy, Luck/Skill Mix) Whenever your team wins a game of Splatoon 3 in the chosen game mode, mark off all weapons that were present on the winning team, including your own. Once a square is marked green, you CANNOT use that weapon again.

Normal Bingo (Medium, Skill Based) Select your weapon each game. Mark off the square containing your current weapon when you win a game. This is traditional speedrunning bingo, which emphasizes skill in selecting the fastest row.

Randomizer Bingo (Hard, Luck Based) Use the randomizer to select your weapon. You must play that weapon until you win a game. When you do, if it is on your card, mark it off and roll again.

Solo Lockout Challenge In this solo variant, mark squares green when you win with the weapon and red when you lose with that weapon. Can you get a bingo without getting locked out?

Past Versions

Version 3.1 This version includes the weapons from the 2.0.0 update.
Version 3.0 This version excludes the weapons from the 2.0.0 update.

Release Date 11/11/2022

Changes:

  • There are two different types of boards now.

    • Normal Mode: This is the most balanced version of bingo to date. The board now has multiple configurations, and each row, column, and diagonal is designed to be tightly balanced.
    Chaos Mode: This is the most unbalanced version of bingo to date. Any weapon can appear in any square without bias or attempts to balance the board in any way!
  • Board generation is now guaranteed and should never result in an infinite loop that will crash your browser.
  • You can now set a seed for the weapon randomizer. By picking the same seed number, all players will experience the same weapon draw no matter what card they are on.
    For example, if you are on card 123 and I am on card 456, we can both get the same randomized order of weapons if we both type the same number in the "Set My Own Seed" field.
    If you both set to "All Weapons" and use the same seed, the order will be the exact same for both players.
    If you both set to "Only weapons on this card" and use the same seed, the weapons themselves will be different, but their places on the board will be in the same order.
  • The supplied weapon box now shows the weapon count as well, which will help you distinguish times when you roll the same weapon twice in a row.
  • Added last/next weapon buttons so you can navigate forwards and backwards.
  • Please note older versions will not be supported going forward, so the new features in version 3 will only apply here.

Version 1

Notes:

  • Version 1 had no weapon bias in card generation. Any square could contain any weapon, without care or consideration put into what other weapons were already in that row.
